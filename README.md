# Projeto de Análise de Complexidade Urbana

Este projeto tem como objetivo analisar a complexidade urbana utilizando dados geográficos e de mobilidade. A análise será realizada utilizando as seguintes bibliotecas Python:
- osmnx
- pandas
- geopandas
- networkx
- folium
- shapely

## Requisitos

Para instalar as dependências necessárias, utilize o seguinte comando:

```bash
pip install -r requirements.txt
```

## Estrutura do Projeto
- `data/`: Contém os dados geográficos e de mobilidade utilizados na análise.
- `notebooks/`: Contém os notebooks Jupyter com a análise e visualizações.
- `src/`: Contém os scripts Python para processamento de dados e análise.
- `README.md`: Este arquivo de documentação.
